# vue-fitd

## Get Start

```vue
<script>
import {FitButton} from 'vue-fitd'
export default {
  template: `<FitButton>Button</Button>`
  components: {
    fitButton: FitButton
  }
}
</script>
```

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
